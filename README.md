# [Game Title]

A 2D Java game developed for the PAOO course.

## Features

- Multiple maps/levels
- Enemies with different behaviors
- Collectibles (KeyCard, Imunosupressors, etc.)
- Health and ability systems
- Animated backgrounds and parallax effects
- Menus: Main, Pause, Settings, Death, Win
- Camera system following the player

## Installation

1. Clone the repository:
   git clone https://github.com/[your-username]/[repo-name].git
2. Open the project in IntelliJ IDEA.
3. Make sure you have JDK 17+ installed.
4. Run the `Game.java` main class.

## How to Play

- WASD to move.
- Space to jump.
- Left/Right to attack.
- F to interact with NPC.
- E to use items.
- Esc to open the pause menu.
- Special abilities:
- Q to use the first ability.
- RMB to use the second ability.
- Shift to Dash.
- Complete objectives to progress through levels.

## Project Structure

- `src/PaooGame/Game.java` - Main game loop and rendering
- `src/PaooGame/Player.java` - Player character logic
- `src/PaooGame/Enemy.java` - Enemy base class
- `src/PaooGame/menus/` - Menu classes (MainMenu, PauseMenu, etc.)
- `src/PaooGame/maps/` - Map and level logic
